# TaskHero Design System

## Brand Colors

Based on the TaskHero logo and cover image:

### Primary Colors
- **Primary Blue**: `#1565C0` - Main brand color
- **Primary Blue Dark**: `#0D47A1` - Hover states, darker elements
- **Primary Blue Light**: `#42A5F5` - Light accents

### Secondary Colors
- **Secondary Gold**: `#FDB913` - Call-to-action buttons, highlights
- **Secondary Gold Dark**: `#E5A812` - Hover states for gold elements
- **Secondary Gold Light**: `#FFCE54` - Light gold accents

### Usage Guidelines

#### Buttons
- **Primary Action**: Gold background with blue text (`bg-[#FDB913] text-[#1565C0]`)
- **Secondary Action**: Blue background with white text (`bg-[#1565C0] text-white`)
- **Tertiary Action**: Blue border with blue text (`border-2 border-[#1565C0] text-[#1565C0]`)

#### Text
- **Headings**: Blue (`text-[#1565C0]`)
- **Body**: Gray (`text-gray-700`)
- **Price/Amount**: Gold (`text-[#FDB913]`)

#### Backgrounds
- **Main Background**: Light gray (`bg-gray-50`)
- **Card Background**: White (`bg-white`)
- **Header**: Blue gradient (`bg-gradient-to-br from-[#1565C0] to-[#0D47A1]`)

## Components

### Header
- Blue background with logo
- Navigation links with gold hover state
- Consistent across all authenticated pages

### Cards
- White background with gray border
- Blue border on hover
- Gold highlights for important information (prices, amounts)

## Pages Overview

1. **Home Page** - Hero section with cover image and features
2. **Login/Signup** - Centered card with logo and brand colors
3. **Dashboard** - Toggle between user tasks and tasker bids
4. **Task List** - Browse all available tasks
5. **Task Details** - View task details, place bids, accept bids
